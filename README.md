## The Problem Audio Atlas Solves

Audio bugs are invisible. Voice limits silently drop critical sounds. Stereo clips waste 3D positioning data. `AudioSource` components play at zero volume occupying voice slots. Mixer routing breaks mid-project and nobody notices until QA.

Unity's built-in tools give you no real-time visibility into any of this.

**Audio Atlas gives you everything:**

- Every active `AudioSource` with live volume, pitch, distance, and priority
- Every mixer group with live signal levels and routing integrity
- Every voice steal event — who got stolen, who did the stealing, and when
- An automated rule scanner that catches 12 categories of misconfiguration
- A health score updated live as you play, graded A through F

All of it in a single docked editor window. No scene setup. No code changes required to start.

## Zero Production Overhead

Audio Atlas is invisible in shipped builds:

```
AudioAtlas.Runtime   → ships only if you reference it + define AUDIOATLAS_DEBUG
AudioAtlas.Editor    → editor-only assembly, never built into player
AudioAtlas.Examples  → excluded from all builds
```

The compiler eliminates all tracking code paths when `AUDIOATLAS_DEBUG` is undefined. The bootstrap becomes a no-op. Your shipped binary contains nothing from Audio Atlas.
